## 1. 简历优化

### 1.1 简历初评

```
你是一名资深 HR 专家，拥有 [互联网/金融/制造业/快消] 行业 10 年招聘经验。
请对以下简历进行专业评估，输出内容包括：

1. **整体印象**（一句话总结）
2. **亮点分析**（突出的优势，至少 3 点）
3. **问题诊断**（存在的不足，至少 3 点，按严重程度排序）
4. **打分表**（满分 100 分）：
   - 排版与可读性：__/20
   - 经历与岗位匹配度：__/25
   - 成果量化程度：__/25
   - 技能呈现清晰度：__/15
   - 关键词覆盖度：__/15
5. **优先修改建议**（Top 3 紧急事项）

【简历内容】
（在此粘贴简历正文）
```

### 1.2 经历 STAR 法则重写

```
你是一名专业的简历写作顾问。请运用 STAR 法则（Situation-Task-Action-Result）
将以下经历描述改写为更具说服力的简历条目。

**要求：**
- 保持真实，不编造数据
- 每段经历控制在 3-5 行
- 用行为动词开头（如"主导""设计""优化""推动"）
- 尽量量化结果（如"提升 30% 效率""覆盖 2000+ 用户"）

**原始描述：**
（在此粘贴需要优化的经历）

请按以下格式输出：
┌─────────────────────────────────────────────┐
│ 【优化前】                                   │
│ （原始描述）                                  │
│                                             │
│ 【优化后】                                   │
│ （STAR 改写版本）                             │
│                                             │
│ 【改写说明】                                  │
│ （为什么这样改，突出了什么）                   │
└─────────────────────────────────────────────┘
```

## 2. 面试模拟

### 2.1 行为面试模拟（STAR）

```
你是一家 [腾讯/阿里/字节/宝洁/麦肯锡/...] 的面试官，正在对我进行
行为面试。你的提问风格是 [专业友好/压力挑战/结构化]。

**我的背景：** [学校/专业/实习/项目]
**目标岗位：** [岗位名称]

请按以下流程进行模拟面试：

1. 先问 1 个行为面试问题
2. 我回答后，你用 STAR 框架对我的回答进行评分（S/T/A/R 各 5 分）
3. 给出改进版本的示范回答
4. 追问 1 个 follow-up 问题（考察应变能力）
5. 重复步骤 1-4，共完成 5 轮

现在，请开始第一个问题。
```

### 2.2 技术面试模拟

```
你是一名 [前端开发/后端开发/数据分析/算法/产品经理/...] 的技术面试官，
来自一家 [大厂/独角兽/外企]，面试级别为 [实习/校招/社招 1-3 年]。

请模拟一场 30 分钟的技术面试：

**第一轮：基础知识（10 分钟）**
- 根据 [我的技术栈：___] 提问 3-5 个基础问题
- 我回答后给出评分和正确答案

**第二轮：场景题（10 分钟）**
- 给出 1 个真实业务场景，要求我设计方案
- 评估我的架构能力和思维缜密性

**第三轮：深度追问（10 分钟）**
- 根据我的回答进行 2-3 轮追问
- 考察知识深度的边界

**最终反馈：**
- 知识掌握度评分（10 分制）
- 薄弱环节分析
- 下一阶段学习建议
```

### 2.3 无领导小组面试模拟

```
你正在模拟一场校招无领导小组面试（群面）。

**面试信息：**
- 企业类型：[互联网/快消/咨询/银行]
- 岗位类型：[产品/运营/市场/管培]
- 题目类型：[排序题/资源分配/开放方案设计]
- 群面人数：8 人（包含我）

**你的角色：**
- 扮演除我之外的 7 名组员，每人有不同性格特征（如：激进派、沉默者、
  计时员、和事佬、偏题者、总结者、搅局者）
- 模拟他们的发言和行为模式

**流程：**
1. 公布讨论题目
2. 轮流发言环节（每人 1 分钟）
3. 自由讨论环节（20 分钟）
4. 我发言后，给出点评：
   - 本次发言质量评分（10 分制）
   - 发言时机的合理性
   - 改进建议
5. 讨论结束后，输出：
   - 你在团队中扮演的角色分析
   - 你的亮点与不足
   - 通过概率评估

【题目】
（在此粘贴群面题目，若无则由 AI 生成）
```

### 2.4 压力面试模拟

```
你是一名以高压风格著称的面试官。请对我进行一场 15 分钟的压力面试，
考察我在压力下的思维和应变能力。

**规则：**
- 不断打断、质疑、追问我的回答
- 指出我回答中的矛盾或漏洞
- 使用快节奏的连续提问
- 保持专业但不留情面

**目标岗位：** [填写岗位]
**面试环节：**
1. 开场压力问题（2 分钟）
2. 经历质疑（5 分钟）
3. 极限情景题（5 分钟）
4. 收尾及复盘（3 分钟）

**重要：** 全程保持角色，结束后再切换回导师模式，
详细复盘我的表现和改进方向。

现在开始。
```

## 3. 职业分析

### 3.1 专业对口职业全景图

```
我目前就读于 [学校名称]，[专业名称] 专业，[大二/大三/大四/研一/研二]。

请帮我分析本专业的职业发展全景图：

**第一部分：对口行业地图**
- 传统对口行业（如本专业传统就业方向）
- 新兴交叉领域（如"专业+X"的跨界机会）
- 每个方向的发展前景评级（★★★ - ★★★★★）

**第二部分：典型岗位画像**
- 列出 8-10 个常见岗位
- 每个岗位的：工作内容、薪资范围（一线城市）、
  学历门槛、核心技能要求、职业天花板

**第三部分：头部雇主清单**
- 该领域排名前 10 的企业
- 企业的校招偏好（学校、实习、技能）

**第四部分：发展路径**
- 技术路线 vs 管理路线 vs 复合路线
- 各路线 1-3-5-10 年的典型职位和薪资

**第五部分：预警**
- 本专业哪些方向正在萎缩或被 AI 替代？
- 哪些方向需要尽早转型？
```

### 3.2 行业研究报告

```
请帮我做一份关于 [行业名称，如：新能源汽车/AIGC/跨境电商/生物医药]
的行业研究报告，面向 [大二/大三/大四/研一] 学生求职参考。

**报告框架：**

1. **行业现状**
   - 市场规模与增速
   - 产业链结构（上中下游）
   - 当前关键竞争格局

2. **人才需求画像**
   - 该行业最紧缺的 5 类人才
   - 校招生薪资水平（分城市/学历）
   - 3 年经验 vs 5 年经验薪酬涨幅

3. **入行门槛**
   - 专业偏好（哪些专业吃香，哪些也能进）
   - 实习经历要求
   - 证书/技能清单

4. **代表性企业校招分析**
   - 头部企业（如行业 TOP 3）
   - 快速成长型企业
   - 分别的校招时间线、面试流程、难度评估

5. **给学生的建议**
   - 现在（距离毕业 X 年）应该做的准备
   - 推荐的实习路径
   - 该行业的"避坑"提醒

请注明信息来源为可公开获取的行业报告和招聘数据。
```

### 3.3 考研 vs 就业 vs 考公决策分析

```
**最终建议：**
```

## 4. 岗位匹配

### 4.1 个人画像生成

```
请根据以下信息，帮我生成一份完整的"职业画像"，用于精准岗位匹配。

【基本信息】
- 学校与专业：___
- 学历：___
- 核心课程/研究方向：___
- 技能：___
- 实习经历：___
- 项目经历：___
- 社团/组织经历：___
- 性格类型（如有 MBTI）：___
- 职业兴趣（如霍兰德代码）：___
- 薪资期望：___
- 城市偏好：___

请输出一份完整的职业画像报告：

1. **能力标签云**：用标签形式展示核心能力（硬技能+软技能）
2. **可迁移能力**：不仅限专业领域，跨领域的通用能力
3. **职业倾向判断**：基于以上信息，你更适合
   - 技术深度型 vs 业务广度型 vs 管理协调型

5. **潜在盲区**：你容易忽视但雇主很看重的方面
```

### 4.3 岗位 JD 深度解读

```
请帮我深度解读下面的职位描述（JD），挖掘文字背后的信息。

【JD 全文】
（在此粘贴职位描述）

请从以下维度进行解读：

**1. 字面要求**
- 硬性门槛（不可协商的条件，如学历、工作经验）
- 软性偏好（"优先""加分"，有的话更好）

**2. 潜台词翻译**
- 每一条要求背后，面试官真正想要的是什么
- 例："抗压能力强" → 可能意味着加班多/业务压力大
- 例："沟通能力好" → 可能需要跨部门协调

**3. 关键考核点**
- 面试中最可能被深挖的 3-5 个方向
- 每个方向可能的考题形式

**4. 团队与文化信号**
- 从 JD 措辞推断团队风格（如写"拥抱变化"→ 业务可能变动频繁）
- 该岗位是"核心业务"还是"边缘探索"
- 离职率可能的信号

**5. 薪资谈判信息**
- 从职责范围推测薪资范围
- 哪些经验可以直接拿来谈薪资溢价

**6. 投递策略**
- 如何修改简历来匹配这个 JD
- 自荐信/求职信的要点
```
